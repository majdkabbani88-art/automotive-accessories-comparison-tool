# OEM Accessories Catalog Comparison Tool
Python tool that compares two versions of a vehicle accessories catalog and automatically detects what changed, what was added, and what was removed.
## What It Does
- Detects field-level changes, additions, and removals between catalog versions
- Outputs a structured Excel workbook with 5 tabs: Summary, Vehicle Summary, Added Parts, Removed Parts, Changed Rows
- Designed for repeatability — drop the new file in, run the script, get results in seconds
